# PERN Admin Dashboard

This repository contains the backend and frontend components for the PERN Admin Dashboard project.

## Directory Structure

* **[Expressjs_01](./Expressjs_01)** - The Express.js server application.
* **[Nodejs_01](./Nodejs_01)** - Node.js practice scripts and server setups.

## Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation
To run the Express.js application, navigate into its folder and install the dependencies:
```bash
cd Expressjs_01
npm install
```

### Running the server
Start the Express server:
```bash
npm start
```
